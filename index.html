03 — Tecnologías

Stack Tecnológico

Herramientas, lenguajes y frameworks aprendidos durante el curso y aplicados en proyectos reales.

Lenguajes de Programación

Python Lenguaje principal para análisis de datos, machine learning y automatización. Usado en el proyecto de clasificación de denuncias y juego de matemáticas discretas.
Java Desarrollo de aplicaciones orientadas a objetos. Aplicado en EcoTrack para implementar estructuras de datos como árboles y listas enlazadas.
JavaScript Desarrollo web frontend e interactividad. Utilizado en Net Zero Green Corridors para visualización geoespacial de datos satelitales.
HTML5 & CSS3 Estructura y diseño de interfaces web. Base del desarrollo frontend en proyectos web y portafolio personal.
SQL Gestión y consulta de bases de datos relacionales. Aplicado en proyectos con MySQL para almacenamiento y recuperación de datos.

Data Science & Machine Learning

Pandas Manipulación y análisis de datos tabulares. Usado para limpiar, transformar y explorar el dataset de 33,052 denuncias de consumidores.
NumPy Operaciones matemáticas y manejo de arrays multidimensionales. Base del procesamiento numérico en modelos de ML.
Scikit-learn Librería principal para machine learning. Aplicada para entrenar modelos de clasificación supervisada en el proyecto de predicción de denuncias.
Jupyter Notebook Entorno interactivo para exploración y análisis de datos (EDA). Permite combinar código, visualizaciones y documentación en un solo lugar.
Power BI Creación de dashboards interactivos e interpretación de resultados. Usado para visualizar métricas y KPIs en proyectos de análisis de datos.

Cloud & Herramientas

Git & GitHub Control de versiones y colaboración en proyectos. Utilizado para gestionar el historial de cambios, ramas y despliegue con GitHub Pages.
AWS / Azure / GCP Exploración de servicios cloud para cómputo, almacenamiento y despliegue de aplicaciones. Aprendidos a través de AWS Educate, Microsoft Learn y Google Skills Boost.
